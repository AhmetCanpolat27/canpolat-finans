# Canpolat Finans

Canpolat Finans is a Streamlit-based finance dashboard project developed for tracking BIST stocks, exchange rates, commodities, and AI-supported market summaries.

## Features
- BIST stock tracking
- Exchange rate and commodity monitoring
- Candlestick chart visualization
- SMA20, SMA50 and RSI indicators
- Intraday volume analysis
- Favorite stock selection
- Google News RSS integration
- AI-powered market summary with Gemini API

## Technologies
- Python
- Streamlit
- yfinance
- pandas
- plotly
- requests
- Google Gemini API
